About The Game:

Scavenger Hunt was created as a way to practice javascript and using the Google Maps API.

The code for this website can be found at my GitHub Page.

How To Play:

To start the game click on the play link. This will take you to the map.

Beside the map you will see a list of items you need to find on the map.

In order to find the items, click on them to revel thier clues.

Items will be indicated on the map with a special marker as seen below.

Just click on the marker to mark the item off as found.
